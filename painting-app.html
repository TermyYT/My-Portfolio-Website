Features

  • Shape of objects, texture of board, and background image can all be changed.
  • Dark mode toggle and lighting slider support.
  • Toggle for timed destruction of paint objects.
  • Button for erasing all drawn contents at once.
  • Toggle for randomized paint object sizes.
  • Toggle for randomized paint object colors.
  • User can choose what color object they want if randomized colors is disabled.
  • Motivational (and sometimes funny) quotes are chosen from a list and displayed at the top of the game window.
  • The board can be rotated to be in portrait or landscape.
  • Main Menu button in bottom left.
  • Working tooltips when hovering over certain UI elements.
  • Drawing instructions below the canvas.

Objective

The goal of this project was to create an interactive, user-friendly painting app using the Unity game engine. I wanted to make it as simple as possible while still delivering a fun product to the players. What started as a very simple project, however, grew to have many wonderful features suggested by friends and family. This was a particularly exciting project because I requested and received feedback on what I should add, what should be fixed, and what needs to be removed. I also got a chance to work with many clever minds over in the Unity Discord server, and it was a humbling experience seeing so many knowledgeable and accomplished developers help each other in creating their games and applications. I borrowed ideas from other developers and implemented them in my own game with unique twists. Additionally, I took to YouTube to find tutorials on how to implement a variety of features.
