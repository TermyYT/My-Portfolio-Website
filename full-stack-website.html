Features

  • See full titles, descriptions, release dates, original prices, discount prices, and current codes for many game titles from the Epic Store.
  • Interact with a user-friendly interface to ensure easy site navigation for all users.
  • Browse a user-friendly version of the website's database filled with mapped API data.
  • Favorite games of your liking and tie them to your account. If you don't want to track them anymore, unfavorite them.
  • Admins can create, list, and assign roles and manage API data. They can also add, update, or remove database entries.
  • Users can see single item views for individual titles, browse the entire database, and make new entries in the database..
  • Large lists are sorted into dynamic pages.

Objective

The goal of this project was to create an easy-to-use website where users could keep track of various game titles from the Epic Games store. The user is able to create an account on the site, head to the game list page, view singular pages for each game, favorite any they like, and refer to their favorites list if they want to keep track of said games. Additionally, long lists are sorted into pages to avoid creating an overwhelming experience for the viewer and there are filters that the user can take advantage of to sort through entries more easily. The website uses Bootstrap for its frontend interface and also takes advantage of front-and-backend validation checks to ensure that proper security measures are present for account safety and to ensure that any form data is submitted correctly.
