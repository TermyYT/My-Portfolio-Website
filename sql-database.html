Features

  • Designed and developed a fully-functional, locally-hosted SQL database system using Oracle SQL.
  • Implemented third normal form (3NF) database normalization to ensure data integrity and optimize efficiency.
  • Designed optimized schema with tables, relationships, and constraints to improve data manipulation and accuracy.
  • Improved data accuracy and query efficiency through rigorous testing and performance optimization.

Objective

The goal of this project was to create a fully-functioning, locally-hosted database using Oracle SQL. The subject matter involved the inner workings of a fictitious car rental service. We were tasked with creating an ER diagram to map the entire system, creating a relational schema to make tabular connections between entities in anticipation of writing the code, and writing the code itself. We were also tasked with ensuring that the database was normalized to Third Normal Form (3NF) at minimum because this would ensure data integrity and optimize efficiency. Through testing, we also found that it was fully functional and ran perfectly even when inputting all the code from scratch. Had this car rental service existed in real life and they were interested in using a database to store their information, this database model would have served as a fantastic starting point for them.
